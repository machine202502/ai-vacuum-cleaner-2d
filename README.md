# Circle Maze

Управляй кругом, доберись до зелёной точки в углу.

Использует [pygame-ce](https://pyga.me/) — Community Edition совместимый с pygame.

## Запуск

```bash
python -m venv venv
venv\Scripts\activate        # Windows
# source venv/bin/activate   # Linux / macOS
source venv/Scripts/activate # Windows bash

pip install -r requirements.txt
python game.py
```

## Управление

| Клавиши | Действие |
|---------|----------|
| WASD / Стрелки | Движение |
| R | Рестарт |
| ESC | Выход |
